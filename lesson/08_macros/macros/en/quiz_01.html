What are macros?




Macros are automatic execution commands, often within standard Microsoft services such as Excel or Word.

True or false: Macros always contain malicious code.



Macros can be harmless if they only contain automated processes, but they can also be misused for malicious code.

Macros are essential to using Microsoft Office.



Macros can be handy for automating processes, but are not essential. For safety reasons, macros should be disabled where possible. Other software solutions or plugins can be used for automation.
