Bei der Arbeit außerhalb des Unternehmens sollten die Daten auf einem Server gespeichert werden.


Statt lediglich lokale Festplatten zu nutzen, arbeiten Sie auf Servern, die ein regelmäßiges Backup erstellen. Nutzen Sie außerdem entsprechende 3-2-1 Backup-Lösungen.

Es ist essentiell, dass auch meine Privaten Geräte zuhause regelmäßig Updates zu Programmen und Betriebssystemen erhalten.



Schützen Sie Ihre Geräte zuhause durch Updates vor Malware. Damit erhöhen Sie auch den Schutz für Ihre Dienstgeräte und/oder das Dienstnetzwerk.
