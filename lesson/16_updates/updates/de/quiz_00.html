Richtig oder falsch?

Updates werden immer automatisch installiert.


Es kann sein, dass Sie erst die automatischen Updates aktivieren müssen. Manchmal sind für zusätzliche Updates Ihre Zustimmung erforderlich.

Updates sind nur einmal im Jahr erforderlich


Aktualisierungen für Software oder Betriebssysteme können öfters erforderlich sein. Auch kurzfristige Updates für aktuelle Bedrohungen können notwendig sein.

Updates schließen Sicherheitslücken.


Der Hauptzweck von Updates besteht darin, relevante Sicherheitslücken in Software oder Betriebssystemen zu schließen.
