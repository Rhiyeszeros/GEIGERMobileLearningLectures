You can protect your data with a secure password.

How to create a secure password:


Use at least 8 characters, but more is always better.

Use upper and lower case letters, numbers and special characters.

It has also been shown that phrases make good passwords. Example:

Use a password that you can easily remember. But be careful not to use personal information that others can know about.

Use a different password for each app and service. A password manager helps you generate good passwords and keep track of them.
